UK Driving Theory Assistant 🚗

A Python-based UK Driving Theory Assistant designed to help learner drivers revise theory questions, test their knowledge, and track their progress.

Features

* Search for driving theory questions and answers
* Browse questions by category
* Fuzzy matching for similar questions
* Interactive multiple-choice quiz system
* Randomized quiz questions
* Quiz scoring and pass/fail results
* Review incorrect answers after each quiz
* Score history tracking
* Average score calculation
* Best score tracking
* Total quizzes taken statistics

How to Run

Clone the repository:

git clone https://github.com/michealsinit-jpg/uk-driving-theory-assistant.git

Navigate to the project folder:

cd uk-driving-theory-assistant

Run the application:

python main.py

Commands

Ask a Question

What is the national speed limit on a motorway?

Start a Quiz

quiz

Show Questions by Category

show signs
show safety
show speed limits

View Quiz Statistics

scores

Exit the Program

quit
